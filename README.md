# Nvim Configuration

Minimal workable [Neovim](https://neovim.io/) configuration. This is what I actually use for *heavy* work as well as simple file editing.

## Profiles

By default, it uses *basic* profile configuration. It's possible to load it in *IDE* mode by switching `NVIM_PROFILE` environment variable to `full`:

```bash
NVIM_PROFILE="full"
```
